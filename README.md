🌡️ Temperature Converter Website (Level 1. Task 3)

📌 Overview

I made a website that helps people convert temperature from one unit to another. The Temperature Converter Website can change temperature values between Celsius and Fahrenheit. It can also do Kelvin if you want.

This project is good for learning about HTML, CSS and JavaScript. It helps me understand how to get information from users how to change the websites content and how to make it work well.

🚀 Features

Here are some things the Temperature Converter Website can do:

* Convert temperature from Celsius to Fahrenheit

* Convert temperature from Fahrenheit to Celsius

* Only let users enter numbers

* Make it easy for users to use

* Show the result away

* Optionally it can also convert temperature to Kelvin

🛠️ Technologies Used

I used HTML5 to make the structure of the website. I used CSS3 to make it look nice.. I used JavaScript to make it work.

📚 What I Learned

I learned how to get information from users when they fill out forms. I learned how to check if the information's correct using JavaScript. I learned how to do math to convert the temperature. I learned how to change what users see on the website.. I learned how to make interactive websites.
